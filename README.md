### <span id="jump_ScreenShot">截屏</span>

&emsp;&emsp;一个功能简单的截屏软件，提供如下功能：

- 支持多屏、高分屏幕；
  
- 支持绘制矩形、绘制文字以及绘制箭头的功能；
  
- 支持撤销操作；
  
- 随时调整截屏区域的位置以及大小；
  
  拖拽选择区域后，可以整体拖拽移动，也可以拖动边框调整大小；

- 提供保存到本地以及剪切板的功能；
